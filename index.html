Condicionales en Excel
¿Qué son?
Una función condicional es aquella que nos permite realizar una acción siempre y cuando se cumpla con una condición determinada por el usuario. Los condicionales se pueden utilizar para modificar el comportamiento de todas las funciones que utiliza Excel
Conceptos necesarios para entenderlos
Los conceptos que yo considero necesarios para usar un condicional en Excel son el uso de signos de comparación como el < o el >
¿Cómo se utilizan?
Para utilizar un condicional, nos debemos ubicar en una celda, y allí escribimos la fórmula del condicional que queremos ejecutar

Funcionamiento de las funciones condicionadas
Sumar si: Suma los números que se encuentran en las celdas seleccioandas si estos cumplen las condiciones deseadas.
Si: Permite realizar comparaciones lógicas entre un valor y un resultado en espera, el resultado final puede ser verdadero o falso.
